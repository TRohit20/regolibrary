# C-0016 - Allow privilege escalation

## Frameworks
* NSA
* AllControls
* ArmoBest
* WorkloadScan
 
## Severity
Medium

apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
spec:
  replicas: 1
  template:
    spec:
      containers:
      - name: nginx
        image: nginx:latest
        securityContext:
          allowPrivilegeEscalation: false # this field should be set to false explicitly
```
